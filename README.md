# ❤️ Heart Attack Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a Heart Attack dataset to identify key factors associated with cardiovascular disease. The analysis focuses on understanding patient characteristics, discovering patterns in health indicators, and generating meaningful insights through statistical analysis and data visualization.

---

## 🎯 Objectives

- Explore the structure of the dataset.
- Clean and preprocess the data.
- Analyze relationships between different health indicators.
- Visualize important trends and patterns.
- Generate meaningful insights related to heart attack risk.

---

## 📂 Dataset Information

The dataset contains medical information collected from patients, including:

- Age
- Gender
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG Results
- Maximum Heart Rate Achieved
- Exercise-Induced Angina
- ST Depression (Oldpeak)
- ST Segment Slope
- Heart Attack Risk (Target Variable)

## 📈 Key Insights

Some important findings from the analysis include:

- Age shows a noticeable relationship with heart attack risk.
- Certain chest pain types are associated with a higher likelihood of heart disease.
- Maximum heart rate varies significantly between patients with and without heart disease.
- Resting ECG results provide useful diagnostic information.
- Cholesterol and blood pressure contribute to cardiovascular risk assessment.
- Correlation analysis highlights relationships among important health indicators.
